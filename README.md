# ⛰️ CumbreGear

CumbreGear es una web especializada en equipamiento para senderismo, trekking, montaña y actividades al aire libre.

El proyecto reúne análisis, comparativas, guías de compra y rutas recomendadas para ayudar a los aficionados a la montaña a elegir el material más adecuado para cada aventura.

## Características

* 🥾 Guías de compra de botas y calzado de montaña.
* 🎒 Comparativas de mochilas y equipamiento técnico.
* 🧭 Reviews de GPS, relojes y dispositivos de navegación.
* ⛺ Recomendaciones de material para camping y trekking.
* 🏔️ Rutas destacadas con información práctica.
* 📱 Diseño responsive optimizado para móviles y escritorio.
* 🔍 Navegación intuitiva y carga rápida.
* 🛒 Integración con enlaces de afiliación de Amazon aún no disponibles.
